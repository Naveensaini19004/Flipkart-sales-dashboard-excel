# Flipkart Sales Performance Dashboard

## Project Overview

This project presents an interactive sales dashboard built in Microsoft Excel using Flipkart sales data. The dashboard helps analyze sales performance, profitability, customer ratings, product categories, regional sales distribution, and payment methods.

## Objectives

- Monitor overall sales performance
- Analyze profit trends
- Compare category-wise sales and profit
- Evaluate regional sales performance
- Track customer ratings
- Understand payment method preferences

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning

## Key KPIs

- Total Sales: ₹75.2M
- Total Orders: 1.0K
- Average Sale: ₹75.2K
- Average Rating: 3.0
- Total Profit: ₹15.0M

## Dashboard Features

### Sales Analysis
- Monthly Sales Trend
- Top 10 Products by Revenue
- Region-wise Sales Distribution

### Profit Analysis
- Monthly Profit Trend
- Profit by Category

### Customer Insights
- Customer Rating Analysis
- Payment Method Distribution

### Interactive Filters
- Region Filter
- Month Filter
- Category Filter
- Customer Segment Filter

## Business Insights

- Electronics generated the highest category sales.
- North region contributed the highest sales.
- Digital payment methods showed strong customer preference.
- Seasonal fluctuations were observed in monthly sales and profit trends.
